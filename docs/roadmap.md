# Roadmap проєкту

## Тиждень 3: Розширення API
* Додати новий endpoint `POST /reserve` для створення бронювання.
* Реалізувати валідацію даних (перевірка, чи вказано ім'я та телефон).
* Оновити клієнтську частину: додати модальне вікно або форму для введення даних.

## Тиждень 4: Авторизація
* Реалізувати базову авторизацію для адміністратора (Login/Password).
* Створити захищений маршрут для адмін-панелі.

## Тиждень 5: CRUD операції
* Реалізувати повний CRUD (Create, Read, Update, Delete) для сутності "Бронювання".
* Додати можливість адміністратору скасовувати бронювання або змінювати статус столика.
