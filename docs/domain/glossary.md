# Глосарій (Ubiquitous Language)

## Основні терміни
1. **Guest (Гість)** — клієнт ресторану, який хоче здійснити бронювання.
2. **Table (Столик)** — фізичне місце в ресторані з певним номером та кількістю місць (capacity).
3. **Reservation (Бронювання)** — підтверджена домовленість про те, що певний столик буде зайнятий певним гостем у визначений час.
4. **Capacity (Місткість)** — максимальна кількість людей, яку може вмістити столик.
5. **Time Slot (Часовий слот)** — проміжок часу, на який можна зробити бронювання (наприклад, 18:00 - 20:00).
6. **Availability (Доступність)** — статус столика, що визначає, чи можна його забронювати на конкретний час.
